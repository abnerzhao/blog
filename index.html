春节自驾回家经历

缘起

买车两年多总体开的比较少，由于是非沪牌平时上班也不开车，就周末偶尔自驾去超市采购、市内公园以及上海周边玩玩。两年多总公里数未超过 2 万公里。新的一年自己心态有一些积极地调整，脑子里有什么想法会积极主动去实践，而非思虑再三然后找各种困难来浇灭它。想做什么就做去做什么，犹犹豫豫不敢迈出第一步最终只能收获惋惜和哀叹。 自驾回家春节在家出行就会灵活方便很多，往年租车费用也不少。自驾回家最担心就是堵车，由于节前假期较多，比往常提早了些。想着应该不会太堵，加上计划路过江西庐山可以游玩过夜，这样路途也不会太辛苦。时机成熟，一切准备就绪那就出发吧。

go build 编译优化

Go 项目使用容器运行时报以下错误：

standard_init_linux.go:207: exec user process caused "no such file or directory"

网上查了一下原因：CGO has to be disabled for alpine.

Dockerfile 中使用的是 golang:1.14-alpine 镜像，Go build 编译参数如下：

GOOS=linux GOARCH=amd64 go build -ldflags "-w -s" -gcflags "-N -l" -mod=vendor -o ./bin/mytest .

显而易见添加 CGO_ENABLED=0 编译之后即可运行成功。CGO_ENABLED 默认是开启的，允许在 Go 代码中调用 C 代码。

go build 编译生产环境下使用的可执行文件一般会加一些优化参数，简单复习一下。
